# The Road to Saki
## 内容
这是一个全新的剧情线MOD，包含了完整的丰川祥子支线。
添加了以下新仪式：
+ 5011001-5011006
添加了以下新事件：
+ 5370001-5370005
添加了以下新角色：
+ 丰川定治（2020015）
+ 三角初音（2020017）
未来可能添加新的boss-“大金毛”和新的结局-“润到瑞士”
你也可以对小祥痛下杀手……但是结局可能无法预料（可能坏档）
Hope you enjoy it!

## 使用方法
+ 首先安装MOD：`Magic Store`
+ 进入`.\rite`，将\rite文件夹中的所有.json文件移入
+ 进入`.\events`，将\event文件夹中的所有.json文件移入
